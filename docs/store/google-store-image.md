## 2. 캡처 체크포인트

- 2개 이상의 덱이 목록에 나타나는 상태로 홈 화면 캡처
- 덱 상세 통계 헤더(총 카드 수, 오늘 복습 수, 숙련율)가 보이는 상태 캡처
- 3D 플립 앞면: 상단 진행 바와 카드 질문 텍스트가 함께 보이도록 구성
- 3D 플립 뒷면: Again/Hard/Good/Easy 4개 버튼 색상 대비 명확하게 표현
- 세션 완료: 정답률 80% 이상 상태, 결과 박스와 버튼이 모두 보이도록 구성
- 카드 추가: 실제 학습 내용이 입력된 상태(예: 단어와 뜻) 권장
- 밝은 테마 기준으로 전체 세트 통일

## 3. 이미지 사양

- 권장 해상도: `1080 x 1920` (9:16)
- 형식: PNG 또는 JPG
- 권장 장수: 8장
- 파일 용량: 이미지당 8MB 이하
