### ko (대한민국 한국어)

**제목** (최대 30자)
`플래시카드 학습`

**간략한 설명** (최대 80자)
`SM-2 간격 반복으로 더 빠르게, 더 오래 기억하세요`

**설명** (최대 4000자)

이미 아는 카드는 건너뛰고, 잊을 것 같은 카드만 정확한 타이밍에 복습하세요.

Flashcard Study는 과학적인 간격 반복(Spaced Repetition) 학습을 빠르고 직관적으로 실천할 수 있도록 설계된 앱입니다.
짧은 이동 시간에도 오늘 복습할 카드만 골라서 학습하고, 즉시 결과를 확인할 수 있습니다.

■ 주요 기능
- 덱 관리: 카드 덱을 자유롭게 만들고 편집하고 삭제
- 3D 플립 카드: 부드러운 Y축 회전 애니메이션으로 앞뒷면 전환
- SM-2 간격 반복: Again / Hard / Good / Easy 4단계 평가로 복습 간격 자동 결정
- 오늘 복습 카드만 학습: 꼭 봐야 할 카드만 추려 학습 효율 극대화
- 진행률 추적: 덱별 숙련율과 오늘 복습 수를 한눈에 확인

■ 학습 방법
- 홈 화면에서 덱을 생성하고 앞면(질문)·뒷면(답변) 카드를 추가합니다
- 덱 화면에서 오늘 복습할 카드 FAB를 탭해 학습 세션을 시작합니다
- 카드를 뒤집고 Again / Hard / Good / Easy 4단계 버튼으로 자기평가합니다
- 세션 완료 후 정답률을 확인하고, 덱 화면에서 숙련율 변화를 체크합니다

■ SM-2 알고리즘
- 오늘 복습 예정인 카드만 모아 세션을 시작합니다
- 카드를 탭해 뒤집고, 얼마나 잘 알았는지 4단계로 평가합니다
- 평가 결과에 따라 다음 복습일이 자동으로 계산됩니다
- 복습 간격 21일 이상, 연속 정답 3회 이상이면 숙련 처리됩니다

■ 데이터 & 오프라인
- 모든 덱과 카드 데이터는 기기 내 로컬 저장 (Hive)
- 오프라인에서도 모든 핵심 기능 사용 가능
- 앱 재실행 시 SM-2 상태(복습 간격, 다음 복습일)가 완전히 복원

■ 광고 안내
- 배너 광고 포함 (완전 무료)
- 학습 세션 완료 후 전면 광고 가끔 노출
- 보상형 광고는 사용자가 선택할 때만 제공
